# ST05 (Performance Trace)

> SQL / RFC / Enqueue / Buffer trace. Captures every DB call (with bind values + execution plan), every RFC, every lock request. Primary tool for finding SQL anti-patterns documented in [[wiki/topics/abap-performance]].
